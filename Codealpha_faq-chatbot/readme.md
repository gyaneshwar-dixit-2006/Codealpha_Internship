# 🤖 FAQ Chatbot

A smart and interactive FAQ Chatbot developed as part of the CodeAlpha Internship Program. The chatbot provides automated responses to frequently asked questions through a simple and user-friendly interface.

# 🚀 Features
Interactive chat interface
Instant FAQ responses
Responsive design
Easy customization
Fast and efficient performance
# 🛠️ Technologies Used
Python
HTML
CSS
JavaScript
🎯 Project Objective

To develop a chatbot capable of answering frequently asked questions automatically, improving user experience and reducing repetitive manual responses.
